# Bundled Fonts

This folder contains fonts bundled directly with All4Laser for reliable text-path generation across systems.

## Included fonts

- `LiberationSans-Regular.ttf`
- `LiberationSerif-Regular.ttf`
- `LiberationMono-Regular.ttf`
- `NotoSans-Regular.ttf`
- `NotoSerif-Regular.ttf`
- `NotoSansMono-Regular.ttf`
- `NotoSansArabic-Regular.ttf`

## License

Bundled fonts in this folder use two licenses depending on family:

- **SIL Open Font License 1.1** (Liberation family)
- **Apache License 2.0** (Noto family)

See full text in:

- `OFL-LICENSE-Liberation.txt`
- `APACHE-LICENSE-Noto.txt`

Both OFL 1.1 and Apache-2.0 are commonly used with GPLv3 projects when distributed as separate third-party assets.
